My API BASEJUMP: Timestamp Microservice

Instrucciones:

Debes pasar un parámetro el que será convertido a tiempo Unix y a una fecha

Si envías tiempo Unix debe regresar ambos resultados, si envías una fecha aplica lo mismo

Si no envias uno de los formatos mencionados anteriormente, retornará nulo

Ejemplo de Uso:

https://fcc-timestamp-microservice-eichgi.c9users.io/1450137600
https://fcc-timestamp-microservice-eichgi.c9users.io/December%2015,%202015

Resultado para ambos casos:

{ "unix": 1450137600, "natural": "December 15, 2015" }
